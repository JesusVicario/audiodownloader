Audio Downloader

Audio Downloader es una aplicación de escritorio desarrollada en Python con Tkinter que permite descargar y convertir audio a formato MP3 desde diferentes fuentes compatibles utilizando yt-dlp.

La aplicación cuenta con una interfaz oscura inspirada en Spotify, barra de progreso en tiempo real, soporte para metadata y descarga automática de portadas.

Características

Interfaz oscura inspirada en Spotify

Descarga de audio y conversión automática a MP3

Barra de progreso en tiempo real

Inserción automática de metadata

Descarga e inserción de portada del contenido

Soporte para playlists completas

Panel de actividad con logs de descarga

Selector de carpeta de destino

Interfaz moderna desarrollada con Tkinter

Requisitos

Para ejecutar la aplicación es necesario tener instalado:

Python 3

yt-dlp

FFmpeg instalado y disponible en el PATH

Node.js recomendado para compatibilidad completa con YouTube

Instalación

Clonar el repositorio

git clone https://github.com/tuusuario/audio-downloader.git

Instalar dependencias

pip install yt-dlp

Ejecutar la aplicación

python app.py
